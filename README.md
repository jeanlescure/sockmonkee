# SockMonkee.io BETA

This is a WIP of an unfederated and ephemeral system of text and file transmission.

Built using Socket.io

Copyright &copy; TicoWebMedia 2016

## Steps to work in:

1. Clone repo.
2. Go in the repo and run:

```bash
npm install
```

3. Run:

```bash
bower install
```

4. Ready to go!

```bash
node server.js
```
